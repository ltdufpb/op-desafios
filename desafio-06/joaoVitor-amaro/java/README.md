# Desafio 06 - Anagrama

# Solução

O programa recebe uma palavra ou frase pela
linha de comando, remove os espaços e converte
as letras para maiúsculas. Também verifica se
a entrada possui apenas letras. Depois,
o programa utiliza backtracking para testar
combinações de palavras que formem anagramas
usando todas as letras da expressão.

# Executar

```
javac Anagrama.java
java Anagrama "oi gente"
```

# Versão

java 21
