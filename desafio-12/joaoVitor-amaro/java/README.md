# Desafio 12 - Potências de 2

## Solução

A solução foi desenvolvida em Java utilizando a 
classe BigInteger, pois os números do arquivo 
podem ultrapassar o limite do tipo long. O programa
ler o arquivo passado com argumento e converte
para bigInteger, em seguida verifica se é potencia
de 2 por meio da operação binária.

## Compilar
```
javac Potencia.java
``` 
## Executar
```
java Potencia d12.txt
```

## Versão

Java 21
