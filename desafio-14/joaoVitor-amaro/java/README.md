# Desafio 14 - Expressão Numerica

## Solução

O programa lê um arquivo `.txt` linha por linha,
separa números, operadores e parênteses da expressão
e converte a expressão infixa para pós-fixa utilizando
pilha. Em seguida, é feito o cálculo de acordo
com a precedência dos operadores.

## Executar

```
javac ExpressaoNumerica.java
java ExpressaoNumerica d14.txt
```

## Versão

Java 21
