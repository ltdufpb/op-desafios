# Desafio 08 - Fração simples

## Solução

O programa recebe uma posição inicial do cavalo
em notação algébrica de xadrez e utiliza backtracking
com a heurística de Warnsdorff para encontrar um
caminho onde o cavalo visite todas as casas do
tabuleiro exatamente uma vez.

# Executar

```
javac Fracao.java
java Fracao frac.txt
```

## Versão
Java 21
