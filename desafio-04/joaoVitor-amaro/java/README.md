# Desafio 04 - Contabilizar Peças de Xadrez

# Solução

A solução realiza a leitura das peças e utiliza estruturas de repetição e condicionais para identificar cada tipo de peça e incrementar seus respectivos contadores.

## Como executar

Compile e executa o programa:

```bash
javac NomeDoArquivo.java


java NomeDoArquivo
```

## Versão

Java 21
